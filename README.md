# Developer Shop

## Install
Install MongoDB
```sh
$ brew install mongodb
```
Run mongo db server
```sh
$ mongod
```
Install project dependencies
```sh
$ npm install
```
Then build and start the project
```sh
$ npm build
$ npm start
```

Open on your browser:
http://localhost:8080/

## Demo
To see the project running on production, go to this link:
http://mazzoni-developer-shop.herokuapp.com/
